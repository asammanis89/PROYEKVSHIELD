📋 INSTALLATION REQUIREMENTS
═══════════════════════════════════════════════════════════════════

1. VirusTotal Account (FREE)
   → https://www.virustotal.com/gui/join-us

2. VirusTotal API Key (FREE)
   → https://www.virustotal.com/gui/my-apikey
   → Limits: 4 req/min, 500/day, 15,500/month

3. Chrome Browser (v88+)

4. Configure background.js:
   const VT_API_KEY = 'YOUR_VIRUSTOTAL_API_KEY';


🚀 QUICK START (3 STEPS)
═══════════════════════════════════════════════════════════════════

Step 1: Edit background.js
   - Open background.js
   - Replace 'YOUR_VIRUSTOTAL_API_KEY' with your actual key
   - Save file

Step 2: Load Extension
   - Open chrome://extensions/
   - Enable "Developer mode"
   - Click "Load unpacked"
   - Select vt-shield folder

Step 3: Test
   - Open any website
   - Hover over any link
   - See security status in tooltip
   - Click extension icon for details
